# Тесты

Запускать из корня проекта:

```
py tests/energy_test.py     # проверка сохранения энергии в ядре (3D)
py tests/test_ui.py         # headless-проверка графического модуля (UI)
py tests/test_scenarios.py  # проверка готовых сценариев (в т.ч. случайных пресетов)
```

Тесты UI используют «пустой» видеодрайвер SDL, поэтому окно не открывается —
тест прогоняет логику и при успехе печатает `ALL OK`.
