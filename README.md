Проект по учебной практике — Gravity_Simulation. Посвящён созданию программы для моделирования движения небесных тел под действием гравитации (задача N тел). Состоит из двух основных частей:

1)  **Вычислительное ядро.** Написано на языке Python. Включает математическую модель закона всемирного тяготения Ньютона и систему классов для хранения данных о телах (`Body`) и управления всей симуляцией (`Simulation`). Для расчёта движения реализованы два метода: метод Эйлера и более точный метод Рунге-Кутты 4-го порядка. Также добавлены проверки входных данных (например, запрет на отрицательную массу) и функция контроля сохранения энергии, чтобы убедиться в правильности работы программы.

2)  **Архитектура и интерфейс для разработчика.** Проект разделён на независимые папки (`Core`, `Utils`), что позволяет писать код чисто и удобно. Предоставлен простой набор команд класса `Simulation`, с помощью которого другой программист может подключить визуализацию: добавлять тела, запускать или ставить на паузу симуляцию, менять скорость времени и получать координаты тел для рисования на экране, не вникая в сложные формулы физики.
