**Проект по учебной практике — Gravity_Simulation.** Посвящён созданию программы для моделирования движения и взаимодействия небесных тел (задача N тел). Состоит из двух основных частей:

1) **Вычислительное ядро.** Включает систему классов для хранения данных о телах (Body) и управления всей симуляцией (Simulation). Поддерживает работу в трёхмерном пространстве. Для расчёта движения реализованы два метода: метод Эйлера и более точный метод Рунге-Кутты 4-го порядка. Также добавлены проверки входных данных (например, запрет на отрицательную массу) и функция контроля сохранения энергии, чтобы убедиться в правильности работы программы.

2) **Архитектура и интерфейс для разработчика.** Проект разделён на независимые модули (Core, UI, Utils). Предоставлен простой набор команд класса Simulation, с помощью которого другой программист может подключить визуализацию: добавлять тела, запускать или ставить на паузу симуляцию, менять скорость времени.
